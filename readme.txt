Плагин для интеграции рассылки с сервисом АвтоОфис

== Description ==

Плагин для интеграции рассылки с сервисом АвтоОфис. Позволяет встроить форму подписки на любую из рассылок АвтоОфис через shortcode или html-код. Поддерживает каналы рекламы. Для работы плагина требуется создать тематическую рассылку в магазине на сервисе АвтоОфис
